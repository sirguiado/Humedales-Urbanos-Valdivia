Clase 1 - Introducción al front-end

Viernes 12 de agosto 2022

La primera clase de Diseño desde el Código tuvimos una pequeña introducción sobre las herramientas técnicas para la implementación de proyectos digitales, también comprender las posibilidades técnicas y particularidades del medio digital, se nos presentaron los lenguajes utilizados en front-end con que seguiremos trabajando: HTML, CSS, JS y el proceso para construir una página web y las herramientas necesarias para esto.

Clase 2 - Introducción a HTML

Viernes 19 de agosto 2022

En esta clase se nos presenta el lenguaje HTML (HyperText Markup Language), se nos explica el concepto de anidación lo que significa que Una etiqueta puede contener a otra etiqueta y también la Indentación, atributos y recomendaciones de escritura para nombrar archivos y carpetas para evitar errores, además se nos muestra la estructura base del index y la semántica en HTML.

Clase 3 - Introducción a CSS

Viernes 26 de agosto 2022

La tercera clase revisamos el lenguaje CSS (Cascading Style Sheets) y su estructura dentro del código, aprendimos como cómo asociar una hoja de estilos a un documento HTML, tuvimos una práctica con GitHub creando un archivo de estilo CSS, crear un commit y subir los cambios al repositorio. Revisamos los selectores CSS y como estos definen sobre qué elementos se aplicará un conjunto de reglas CSS, selectores elementales, selectores por atributo, especificidad y herencia.

Clase 4 - Variables, Tipografía, Colores, Íconos

Viernes 2 de septiembre 2022

Para la cuarta clase seguimos revisando el lenguaje CSS, estudiamos como cambiar la tipografía y definir la familia de fuentes que se va a utilizar, revisamos la utilización de tipografías específicas (webfonts), vimos tamaño y espaciado del texto, mayúsculas y minúsculas además como alinear los párrafos con sus diferentes formas, estudiamos el color, trasparencia y realizamos un ejercicio básico de CSS.

Clase 5 - Diagramación

Viernes 9 de septiembre 2022

Esta clase, vimos cómo diagramar documentos HTML. Desde el principio, hicimos un ejercicio práctico que consistía en intentar diagramar una página copiando la página presentada a partir de los conceptos mostrados anteriormente en las clases. Aprendimos sobre características de CSS Grid, vocabulario fundamental y definir una grilla: identificando líneas, vimos posiciones y un ejercicio en CodePen, entre varios otros conceptos.

Clase 6 - Diagramación II

Viernes 23 de septiembre 2022

En esta clase continuamos viendo diagramación, para colocar y distribuir los elementos a lo largo de una página. Esto se puede solucionar con diferentes métodos de diagramación. Primero comenzamos con contenedor de grillas, lo que transforma un elemento HTML en una cuadricula. Dentro de esta cuadricula se colocan los elementos (hijos) que componen la página, y se definen posiciones para estos. Tambien se usa el método de Flexbox, o caja flexible, un método de layout para distribuir el espacio entre los elementos de una interfaz, definir dirección, dimensiones, ejes y orden, y mejorar la alineación de estos.

Clase 7 - Sistemas de diseño y frameworks CSS

Viernes 30 de septiembre 2022

El Sistemas de diseño es elaboración de una propuesta que no tiene que ver solamente con la estética visual, sino que es todo un concepto de cómo se dispone una marca, tono musical, movimiento, interacciones en la pantalla, etc. Como se declara el diseño desde esos lugares, los principios de diseño que se abordan. IMB Design Language - Tiene la declaración de elementos de interfaz, grillas, iconografía, etc. Material Design – sistema de diseño que propuso Google para construir todos sus productos digitales. Sistema de diseño con movimientos, por capas, profundidades, aspectos sensibles visualmente. Orientado al desarrollo, tiene distintos frameworks. El Framework nos entrega diseños para tener una base para diseñar. Ej, Bootstrap. Tiene elementos ya definidos. Elementos que hemos visto hasta ahora y hemos hecho paso a paso, en esta pagina ya estan hechas. Está todo documentado, para tener inmediatamente un resultado en html. Permite avanzar rápido en la implementación, y tiene compatibilidad con la mayoría de las plataformas. Es de código abierto. Trabajan con buena practica de código. Son modulares, permiten utilizar ciertos aspectos y extender sus estilos.

Clase 8 - Imágenes + Accesibilidad / Seo

Viernes 14 de octubre 2022

Revisamos Imganes y accesibilidad, con el uso de Imganes vimos los formatos comunes para paginas web como: jpg, png, gif, svg y webp, vimos la Optimización de imágenes ya que la velocidad de descarga de nuestras imágenes impacta en la accesibilidad y SEO de nuestros sitios. Imágenes adaptables que es entregar un archivo de imagen diferente dependiendo de características del dispositivo, utilizando los atributos srcset y sizes. Revisamos las técnicas básicas para la accesibilidad entre codigo, diseño, contenido y interactividad. Por ultimos estudiamos SEO Search Engine Optimization (optimización para motores de búsqueda).
